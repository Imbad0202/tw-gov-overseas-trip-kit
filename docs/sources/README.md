## 主要法規

### 1. 國外出差旅費報支要點

- **版本**：114.05.13 修正
- **文號**：院授主預字第1140101390號函
- **URL**：https://law.dgbas.gov.tw/LawContent.aspx?id=FL017584
- **說明**：計算機關出差旅費（交通費、住宿費、雜費、日支生活費）的核心規則，本工具計算邏輯以此為準。衝突時以此版本為最終依據。

### 2. 生活費日支數額表

- **版本**：114.10.31 修正、115.1.1 生效
- **文號**：院授主預字第1140103430號函
- **URL**：行政院主計總處公開資料（https://www.dgbas.gov.tw/）
- **說明**：各國每日生活費補助上限，**逐年更新**。本工具不內建表格數值；使用者帶入時應以當下最新版本為準，自負其責。

### 3. 國外出差旅費報支要點解釋彙編

- **版本**：107.4 修編
- **URL**：行政院主計總處（https://www.dgbas.gov.tw/）
- **說明**：對要點各條文的官方問答解釋，本工具計算邏輯設計時作為參考。

### 4. 行政院出國報告綜合處理要點

- **版本**：107.06.20
- **附件**：附件一（出國報告格式）、附件二（費用核銷明細）
- **URL**：行政院公開資料（https://www.ey.gov.tw/）
- **說明**：本工具 HTML、DOCX、Excel 渲染輸出格式的主要依據。
