# 出差旅費個案報支指引（延返 · 艙等 · 保險等）

本指引就國外出差旅費報支的常見個案提供**方向性**參考，協助承辦初步判斷應循哪條規則、該備哪些文件。

**本工具不做個案認定、不構成法規解釋或報支保證。** 每個主題只講「往哪個方向想、該確認什麼」，不下最終結論、不給可照抄的金額或比例。凡標「**需向官方確認**」處，一律至主計總處法規查詢系統核對現行條文，或向貴機關主計確認。本指引不內建函釋個案庫、不引用特定函釋文號；需正本者請至主計總處取得。

> 法規版本：本指引方向對齊國外出差旅費報支要點現行架構。實際報支以主計總處現行要點及貴機關核定為準。

## A. 人員適用

**方向**：報支要點的適用對象是「經機關首長核准出國執行任務之公務人員」。

- 聘用 / 約僱人員：方向是**可比照公務人員**報支（相當薦任 / 委任）。→ **需向官方確認**。
- 工友（含技工、駕駛）、非公務法制進用之臨時人員：方向是**非報支要點適用對象**、不宜派遣出國考察。→ **需向官方確認**。
- 以「公假」登記而非「出差」性質者：方向是**不得報支**國外旅費（須符合出差定義：有年度出國計畫、預算，並經首長核准行程日數）。→ **需向官方確認**。

## B. 交通費（機票 / 陸運 / 機場往返）

**方向**：交通費涵蓋飛機、船舶、長途大眾陸運工具；市區車票屬零用費（含在生活費內），不另報交通費。

- **機票單據**：機票部分方向是需檢附「機票票根或電子機票」＋「購票證明單或旅行業代收轉付收據」＋「登機證存根（或足資證明出國事實之文件）」。單據不齊須補正。→ **需向官方確認**。
- **本國籍航空**：方向是原則搭本國籍航空；例外（客滿、無航班、轉機等待逾時、無法銜接等）須先填單經首長核定改搭外籍，否則票款可能不予核銷。→ **需向官方確認**。
- **廉價航空**：方向是機票內含之必要項目（限額行李、餐飲、非特殊需求選位、手續費）可比照一般航空報支，但總額應低於一般航空；其餘可選項目（毛毯、眼罩等）多不予核銷。→ **需向官方確認**。
- **租車 / 計程車**：方向是特殊需求確有必要者，租車費多在禮品交際及雜費項下報支。→ **需向官方確認**。
- **機場往返交通費**：方向是比照國內出差辦理；若已由補助方免費接送而個人選擇不用，衍生費用可能自行負擔。→ **需向官方確認**。

**工具端**：機票金額屬 B 類 `manual_items`，由承辦填實際票價，工具不判。

## C. 搭機艙等

**方向**：艙等依**職等 / 職務**判定，工具不自動判，由承辦依報支要點第 5 點認定。

- 部長級 / 特使：得頭等；次長級、簡任第十二職等以上領全額主管加給者：得商務；其餘：經濟（標準）艙。→ 具體對應 **需向官方確認**。
- 得乘商務艙者，若該航線**僅有頭等與經濟兩種艙等**（無商務）→ 方向是**只能坐經濟艙**（無「改坐頭等」之規定）。→ **需向官方確認**。
- 兼任 / 權理較高職等主管職務、支領該職等主管加給者，能否適用該職等艙等 → 視**出國事由是否為執行該職務所需**，屬個案。→ **需向官方確認**。
- 「豪華經濟艙」「全價票經濟艙」：方向是規定無此艙等，僅得以**經濟（標準）艙**價格報支。→ **需向官方確認**。
- 地方政府首長（直轄市長 / 縣市長）：方向是可能有準用部長級 / 次長級標準之規定。→ **需向官方確認**。

**工具端**：機票金額屬 B 類 `manual_items`，工具不判艙等。

## D. 生活費日支

**方向**：日支費不必檢據（覈實住宿等特定情形除外），劃分為住宿 70% / 膳食 20% / 零用 10%。**日支數額表本工具不內建**，由使用者帶入當年度官方版本。

- **未列載城市**：方向是填該國「其他」欄數額（不擴大至都會區）。**未列載國家**：填地理最近國數額。→ **需向官方確認**。
- **一日跨多地區**：方向是以**當日留宿地**之數額為準，不得重複。→ **需向官方確認**。
- **交通工具歇夜 / 返國當日**：方向是按起飛地日支數額之限額比例報支（歇夜屬「供宿不供膳」；返國當日另有限額）。兩者意義不同、不可混用（見〈計算器 scope 邊界〉B-6）。→ **需向官方確認**。
- **委外代辦（旅行社包辦膳宿）**：方向是個人多僅能報零用費部分，住宿膳食依決標金額列報，差額視為機關撙節。→ **需向官方確認**。

**工具端**：`per_diem_base` 由使用者填當年度數額；歇夜 / 返國當日以 `host_provided` / `is_return_day` 旗標反映（見 SKILL.md）。

## E. 供膳宿與現金津貼扣補

**方向**：主辦或其他來源提供膳宿 / 現金津貼時，已供部分不得重領，僅報未供部分或補足零用費。

- 供膳宿：方向是僅報 / 補足零用費至 10%。
- 供膳不供宿：方向是住宿 70% ＋ 零用補足。
- 供宿不供膳：方向是膳食 20% ＋ 零用補足。
- 供膳未達三餐：方向是早 / 中 / 晚以 4% / 8% / 8% 補足未供餐。

以上比例對齊報支要點第 9 點；**是否符合各情形、現金津貼如何認定屬個案** → **需向官方確認**。

**工具端**：由 `host_provided`（`board_and_lodging` / `board_only` / `lodging_only` / `none`）、`meals_not_provided`、`cash_allowance_usd` 表達，計算器自動補足（見 `calc/per_diem.py`）。

## F. 延後返國 / 不可抗力滯留

**方向**：延返能否報支生活費，關鍵在「**是否可歸責於出差人員**」與「**是否經核准**」。

| 情形 | 方向 |
|---|---|
| 非可歸責事由（颱風、氣候、班機延誤等意外阻滯） | 報支要點**第 12 點**訂有相關規定：意外事故阻滯致超出預定日數，經提出確實證明、並經機關首長核准者，得按日報支。是否符合、如何認定 → **需向官方確認**。 |
| 可歸責事由（旺季訂不到票、私人因素延後） | 一般不屬第 12 點意外事故，延返期間多不得報支。是否例外 → **需向官方確認**。 |

- **延返後「返國當日」如何認定**、跨國際換日線的日期計算，屬個案 → **需向官方確認**。
- **延返且非從出差地回國**（改赴第三地）：交通費方向是按必經順路計算，超額 / 私人段多自行負擔。→ **需向官方確認**。

**工具端**：延返日獲核准後，該日 `segment` 標 `approved_extension: true` 才算進日支費；未標預設為 0（見 SKILL.md〈計算器 scope 邊界〉與 `calc.per_diem` 之 `approved_days` / `approved_extension`）。

## G. 手續費 / 保險 / 行政費 / 雜費禮品 / 匯率

**手續費**（第 13 點）：方向是護照費、簽證費、黃皮書費、預防針費、結匯手續費、機場服務費，檢據覈實報支。護照相片費多不屬護照費範圍。→ **需向官方確認**。

**保險**（第 14 點）：方向是出差人員**應辦理保險**，保險費檢據**覈實報支**——即投保是規範預期動作、保費可報。保險項目與保額由行政院另定（屬法定內容，含意外死亡 / 殘廢 / 傷害醫療 / 航空旅行 / 疾病住院 / 兵災等，赴不同地區另有規定）。現行項目與保額（會隨承作保險公司與年度調整）→ **需向官方確認**。

> 「出國前沒投保會怎樣」：實質後果是放棄一筆本可覈實報支的保險、及一層法定保障；是否構成程序疏漏屬各機關差旅管理與內控 → **需向貴機關確認**。保險**理賠**（颱風延誤等）不在本工具與報支要點範圍，依投保之**保單條款**與保險公司認定。

**行政費 / 信用卡手續費**：方向是註冊費等經主辦限定須以個人信用卡支付者，刷卡手續費得併同報支；未限定則選費用最低方式。→ **需向官方確認**。

**雜費 / 禮品交際費**：方向是率團司處長級以上者按職級有上限額（檢據）；非隨團者按報支要點所定每人每日定額總額內（檢據）。具體額度 → **需向官方確認**。

**匯率**：方向是出國前未結匯者，以出國前一日臺灣銀行賣出即期美元參考匯價為準；已結匯者依結匯匯率。→ **需向官方確認**。

**工具端**：以上多屬 B 類 `manual_items`，由承辦填實際金額 / 依機關規定核可。

## H. 進修研習 / 受刑返國 / 跨修法

以下情形**明確不在本工具計算範圍**，需承辦人工判斷（見 SKILL.md〈計算器 scope 邊界〉）：

- **進修 / 研究 / 實習 / 受訓人員**：適用「派赴國外進修、研究、實習人員補助項目及數額表」（另一套補助制度，非本要點日支），本工具不涵蓋。→ **需向官方確認**。
- **駐外人員**（第 10 點）：另有規定，不得直接套用本工具。
- **受刑期限返國**（第 18 點）：涉個案核定，本工具不處理。
- **跨修法分段**（第 23 點）：出差橫跨要點修正前後，由承辦手動分段填入各 `segment.per_diem_base`。
